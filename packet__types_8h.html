Detailed Description

Contains the types and functions required for constructing packets.

This header file contains all function prototypes for packet_types.c, as well as the enums specified in the CUInSpace packet encoding spec for device addresses, block types and sub-types.

All packet types are tightly packed in order to fit in their exact specified byte size. Contents of the packet types can be accessed individually or as a byte array.
